Generating docs

This published version of the docs can be refreshed by these commands:

  1. Prepare one off

    dotnet tool restore git clone https://github.com/dsyme/DiffSharp ../DiffSharp-docs -b gh-pages --depth 1

  2. To Iterate on Literate Docs

    dotnet fsdocs watch --eval

    open http://localhost:8901/index.html

  3. To Iterate on API Docs (requires evaluation off since DLLs get locked)

    dotnet fsdocs watch

    explorer http://localhost:8901/reference/index.html

  4. Build Final and Release

    dotnet build dotnet fsdocs build --eval --clean --output ../DiffSharp-docs bash -c "(cd ../DiffSharp-docs && git add . && git commit -a -m doc-update && git push -f https://github.com/dsyme/DiffSharp gh-pages)"
